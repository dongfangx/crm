## 需求描述
1. 记录客户信息(包括级别信息，多个联系人信息)
2. 记录公司联系客户的反馈信息
3. 记录客户发起的事件
5. 记录客户交易流程信息，包括（初期沟通客户，签合同，后期维护，
3. 维护，产品，员工，客户信息
4. 数据统计分析
	4.1 统计产品的销售情况
	4.2 统计员工销售产品情况
	4.3 统计客户采购情况
	4.4 统计维护

## 技术架构
1. Java：Nutz框架

3. 静态模板框架：Beetl
4. 后台UI框架：B-JUI
5. 数据库：MySql
6. 后台服务器：阿里云centos6
